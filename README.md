基于 Python 的网络安全综合防护系统
项目简介

本项目为《网络安全与系统设计》课程设计实验成果，采用 Python 编程方式实现一个轻量级的软件化网络安全综合防护系统。
系统不依赖专用硬件防火墙或商业安全设备，而是在通用 Linux 操作系统环境下，通过脚本形式实现网络访问控制、入侵检测、主机安全加固以及数据加密保护等核心安全功能。

项目主要目标是验证网络安全相关原理与策略的可行性，具备良好的教学实验价值与可复现性。

系统功能概述

本系统由四个相互独立但逻辑关联的安全模块组成，对应课程评分标准中的四个技术实现方向：

1. 防火墙访问控制模块

通过 Python 调用系统防火墙机制（iptables），实现基于规则的网络访问控制。
该模块遵循最小权限原则，默认拒绝所有未授权访问，仅允许明确配置的端口和协议通过，用于模拟基础防火墙防护能力。

主要功能包括：

设置默认拒绝策略

放行必要服务端口（如 SSH）

限制异常 ICMP 请求

提供基础的网络边界防护能力

2. 入侵检测模块

入侵检测模块采用日志分析与行为统计的方式实现，通过 Python 脚本分析网络连接或访问日志，识别异常高频连接、端口扫描等典型攻击行为。

该模块用于模拟 IDS的核心工作机制，而非深度包检测。

主要功能包括：

统计单位时间内的连接次数

检测疑似端口扫描或暴力连接行为

输出告警信息供管理员分析

3. 主机安全加固模块

主机安全模块通过 Python 自动化脚本完成对系统基础安全配置的检查与加固，降低因默认配置不当带来的安全风险。

该模块重点体现“安全基线”与“最小暴露面”理念。

主要功能包括：

检查用户密码复杂度策略

识别并提示高风险账户配置

关闭或提示不必要的系统服务

提升主机自身的防护能力

4. 数据安全加密模块

数据安全模块基于对称加密算法，实现对敏感文件的加密与解密操作，验证数据在存储状态下的机密性保护机制。

该模块用于体现密码学在实际安全系统中的基础应用。

主要功能包括：

对指定文件进行加密存储

支持解密还原原始数据

防止明文数据被未授权访问
